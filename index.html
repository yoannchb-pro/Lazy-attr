❤️ Lazy-attr.js (click to download)

Example of an image which this lazy load and which when scrolling launches the animation even after having already performed once the animation. lazy-src="image source" lazy-animation="class animation" lazy-reset

Example of an image which this lazy load and which when scrolling launches the animation but only once. Then it is no longer taken into account. lazy-src="image source" lazy-animation="class animation" lazy-animation-time="time in seconde"

Example of an image which this lazy load and which during scrolling several times. Then it is no longer taken into account. lazy-src="image source" lazy-animation="class name" lazy-reset

Example of an embed video which launches during the click and which executes an animation during the scroll. lazy-embed="url of embed video" lazy-animation="class animation"

Example of an image which this lazy load and which during scrolling several times. Then it is no longer taken into account. This time, if the animation does not launch with a class but a style. lazy-src="image source" lazy-animation="class name" lazy-reset

Allows you to perform an animation with a gap delay between each frame here in the example below. lazy-animation-delay="time in seconde"

You will find below the list of events. To add an animation just do: lazy-animation="{animation name}-animation"
